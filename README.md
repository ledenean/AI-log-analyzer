# AI-log-analyzer
Overview
This is a Python-based tool that leverages generative AI to analyze application logs, identify recurring errors, and summarize incidents. It produces both internal technical reports and customer-facing summaries, helping technical teams troubleshoot issues faster and understand system health and user impact.

Features
- Parses application log files to detect errors and patterns.
- Summarizes root causes for faster technical investigation.
- Generates structured internal summaries for documentation.
- Creates customer-facing incident summaries that translate technical issues into clear language.
- Assesses customer impact and suggests preventative improvements.

Tech Stack
- Languages: Python
- Libraries/Tools: OpenAI API, Streamlit
